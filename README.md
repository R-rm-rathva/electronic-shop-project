

Welcome to **ElectroMart Pro**, a modern electronic shop landing page designed for a seamless user experience. This project was developed as a part of our web development portfolio.

## ✨ Key Features
* **Smart Gadget Comparison:** Side-by-side technical specification comparison for all products.
* **Dynamic EMI Calculator:** Real-time monthly installment estimates for high-value electronics.
* **Live Stock Indicator:** Visual progress bars showing remaining inventory to create urgency.
* **Dark Mode Support:** Toggle between light and professional dark themes.
* **Floating Interactive Cart:** Smooth UI notifications when items are added to the cart.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties (Variables), Flexbox, Grid, and Animations.
* **JavaScript:** DOM Manipulation and logic for the comparison engine.
